Network Security Monitoring

Project Optimus

Optimus helps to monitor and analyse the Enterprise Network traffic to generate insightful information and this can be used as a reactive and proactive solution. We have improved enrichment capability by merging multiple detection techniques like signature based detection, hash based detection etc. Proven & tested solution for live traffic capture and historical traffic analysis..

Features

  • EY Optimus help to capture live enterprise network traffic. Further it helps to analyze the raw pcap files and individual packets. Network intrusion detection from pcap files is now easy with Optimus.

    1. Network Security Monitoring

    2. PCAP Trace Back

    3. PCAP Analysis

    4. Network Intrusion Detection
